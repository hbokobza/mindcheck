MindCheck — Interrogatoire clinique guidé

Décrivez votre situation librement. Vous pouvez compléter votre texte par des documents joints. L’objectif est de transmettre au moteur suffisamment d’éléments pour produire une première synthèse structurée et exploitable.

Entrée utilisateur

Le mode “oral” n’enregistre pas automatiquement le micro dans cette version. Il sert à indiquer au backend la nature de l’entrée. Pour un vrai mode micro, il faudra ajouter la capture/transcription ensuite.
Plus le matériau clinique est précis, plus la synthèse sera utile.
Vous pouvez joindre des documents complémentaires : notes, textes, captures, PDF, images. Dans cette version, les fichiers sont transmis au backend en base64.
Cette page front ne remplace pas vos fichiers de logique serveur. Elle envoie simplement les données à /api/chat. Vos modules lib/... existants restent utilisés côté backend.

Réponse du moteur

API : /api/chat Front simplifié Compatible Vercel
La réponse apparaîtra ici.
Si rien ne s’affiche, le problème vient en général de l’un de ces points : format JSON envoyé, erreur dans api/chat.js, timeout, ou réponse backend non conforme.
